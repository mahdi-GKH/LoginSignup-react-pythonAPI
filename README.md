# LoginSignup-react-pythonAPI

this is a simple Login and Signup from react(Axios) and python(API with flask)

# install and run :
  - cd LoginSignup-react-pythonAPI
  - npm install
   - npm start
   - pip install -r requirements.txt > LoginSignup-react-pythonAPI/Python
   - python app.py
